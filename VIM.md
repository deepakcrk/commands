**Incrementing column numbers**

```
vector[0] = 1;       vector[0] = 1;
vector[0] = 1;       vector[1] = 1;
vector[0] = 1;  -->  vector[2] = 1;
vector[0] = 1;       vector[3] = 1;
vector[0] = 1;       vector[4] = 1;
```

```
Put cursor on topmost zero, select column with Ctrl-v, then g Ctrl-a
```
